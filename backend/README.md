# Backend Setup

## Directory Structure

```
backend/
│
├── bin/
│   └── www.js
│
├── models/
│   └── .gitkeep
│
├── routes/
│   └── .gitkeep
│
├── app.js
├── package.json
├── package-lock.json
├── populate.js
└── README.md
```

## Install Dependencies

```
npm install
```

## IMPORTANT

```
Make sure the Backend server port has Visibility Public
```

## Start Backend Server

```
npm start
```
